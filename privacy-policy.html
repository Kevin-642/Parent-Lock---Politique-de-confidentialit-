Politique de confidentialité — ParentLock

Dernière mise à jour : 20 juillet 2026

ParentLock est une application Android de contrôle parental. Cette politique explique quelles informations sont utilisées, pourquoi elles le sont et où elles sont conservées.

1. Informations utilisées

  • Applications installées : ParentLock lit les applications lançables afin que le parent puisse choisir celles qui doivent être protégées.
  • Service d’accessibilité : ParentLock observe uniquement le changement d’application au premier plan pour afficher l’écran de verrouillage lorsqu’une application choisie est ouverte. Le contenu des écrans, les messages, les saisies et les mots de passe ne sont pas collectés.
  • Accès aux statistiques d’utilisation : ParentLock utilise Usage Access pour calculer le temps d’utilisation quotidien des applications placées dans le coffre.
  • Biométrie : l’authentification est réalisée par le composant Android BiometricPrompt. ParentLock ne reçoit ni ne stocke les empreintes, le visage ou le code de déverrouillage.
  • Configuration familiale : les profils, horaires, applications verrouillées et préférences sont stockés localement sur l’appareil.

2. Utilisation des informations

Ces informations servent exclusivement à appliquer les règles parentales configurées par le parent : identifier l’application ouverte, vérifier la plage horaire, calculer la limite quotidienne et demander une authentification avant l’accès.

3. Partage et transfert

ParentLock ne vend, ne loue et ne partage pas ces informations avec des tiers. L’application ne transmet pas la configuration familiale, la liste des applications ou les statistiques d’utilisation à un serveur distant dans cette version.

4. Conservation et suppression

Les données restent dans le stockage privé de l’application. Elles sont supprimées lors de la désinstallation de ParentLock. Le parent peut également retirer à tout moment une application du coffre ou modifier les profils et horaires.

5. Enfants et contrôle parental

ParentLock est configuré par un parent ou un responsable légal. L’application ne crée pas de compte enfant et ne collecte pas directement de données personnelles auprès des enfants.

6. Sécurité

La configuration est conservée localement dans l’espace privé Android de l’application. L’accès aux fonctions parentales sensibles est protégé par l’authentification de l’appareil. Aucune sécurité logicielle ne peut toutefois empêcher tous les contournements propres au système Android, notamment le mode sans échec ou les changements d’utilisateur.
